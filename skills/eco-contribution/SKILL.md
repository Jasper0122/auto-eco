# Eco-Contribution：贡献声明打磨

## 你的任务

基于已完成的文献调研和实证结果，找出本研究相对于现有文献的**真实增量**，输出严格的贡献声明。

贡献声明的目标读者是审稿人和 Editor，他们最关心：**这篇文章有什么是已有文献没做到的？**

## 参数解析

- $ARGUMENTS：项目目录路径
  - 该目录下应已有 `ECO_LIT.md` 和 `ECO_RESULTS.md`

---

## 步骤 1：提取本文核心要素

读取 `ECO_LIT.md`，提取：
- 研究主题（X → Y，具体政策事件）
- 识别策略（DiD/IV/RDD等）
- 样本（国家/地区/时间段/单元类型）
- 机制渠道（已检验的渠道变量）
- 异质性维度

读取 `ECO_RESULTS.md`，提取：
- 主系数方向与显著性
- 效应量（占均值%）
- 哪些稳健性通过
- CS-DiD 结果（若有）
- 最显著的异质性发现

---

## 步骤 2：搜索最近似的已有文献

对每个贡献维度（因果识别、情境、机制），各执行 1-2 次 WebSearch 确认"已有文献的边界在哪里"。

搜索模板：
```
[研究主题关键词] causal identification [DiD/IV/RDD] AER QJE JPE "difference-in-differences"
```
```
[具体政策名] economic impact [X效应] China panel data
```
```
[机制变量] mediating channel [Y] empirical evidence
```

对每次搜索，记录：已有文献的样本范围、识别策略、研究维度，以便精确定位本文的增量所在。

---

## 步骤 3：识别增量类型

从以下5种增量类型中判断本文属于哪几种（可多选）：

| 增量类型 | 判断标准 | 强/弱 |
|---------|---------|------|
| **因果识别升级** | 已有文献用OLS/相关性，本文用DiD/IV/RDD | 强 |
| **情境填补** | 已有文献针对其他国家/地区/政策，本文填补中国/特定政策的空白 | 中 |
| **机制揭示** | 已有文献记录了 X→Y，本文额外打开了"为什么"的黑箱 | 中 |
| **异质性发现** | 已有文献报告平均效应，本文发现对特定子群效应差异显著 | 中 |
| **方法改进** | 本文解决了已有文献未处理的 Staggered DiD 偏误/测量误差等问题 | 强（若适用） |

**弱增量（不应独立作为贡献点）**：
- "据我们所知，这是第一篇研究X对Y的论文"（大概率假命题）
- "本文数据集更新/更大"（不是贡献，是工具）
- "本文证实了已有发现"（这是无增量的复现）

---

## 步骤 4：起草3条贡献声明

每条贡献声明结构：**[本文做了什么] + [已有文献的边界] + [因此本文的增量]**

字数限制：每条 ≤ 80 字（中文）

**格式模板**：
```
第一，[本文识别策略] 为 [研究问题] 提供因果证据。
已有文献多依赖 [OLS相关/跨国截面]，难以排除 [具体内生性来源]。
本文利用 [具体政策/准实验] 构造外生变化，发现 [主系数方向与量级]。

第二，本文揭示 [机制/异质性]。
[具体机制/分组发现] 表明 [经济逻辑]，
弥补了已有文献对 [该机制/子群] 关注不足的缺口。

第三，本文为 [政策/情境] 提供直接证据。
[具体情境的独特性]，使本文的发现对 [政策含义/后续研究] 具有参考价值。
```

**检查清单（每条都要能回答"是"）**：
- [ ] 有具体数字支撑（系数、效应量）？
- [ ] 点名了被超越的已有文献（作者年份）？
- [ ] 避免了"据我们所知第一篇"这类无法证伪的表述？
- [ ] ≤ 80字？

```markdown
## 编辑/审稿人可能的质疑

1. **"与 [XX文献] 的区别是什么？"**
   → [应答：具体指出识别策略/情境/机制的差异]

2. **"贡献X够发表吗？"**
   → [应答：结合已有文献档次和本文增量，给出坦诚评估]
```

---

## 注意事项

- 贡献声明不是摘要，不要重述研究设计或数据——只说**增量**
- 若主系数不显著，贡献点应转向：高质量识别证明"效应小/为零"也是贡献（精确零效应），或从机制/异质性中找显著发现
- 若文献搜索发现本文与已有研究高度重合（≤1年前发表的AER/QJE同题文章），如实告知用户，而非硬撑贡献
