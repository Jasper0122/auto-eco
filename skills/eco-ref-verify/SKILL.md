# Eco-Ref-Verify：引用真实性验证

## 你的任务

读取项目目录下的 `ECO_LIT.md`，对其中每一条参考文献逐一搜索核实，输出 `ECO_REFS_VERIFIED.md`。

## 参数解析

- $ARGUMENTS：项目目录路径（如 `E:\my_project`）
  - 该目录下应已有 `ECO_LIT.md`

---

## 步骤 1：提取所有引用

读取 `ECO_LIT.md`，从文献脉络、推荐识别策略、机制/异质性支撑等各处提取**所有引用**。

对每条引用，解析：
- 第一作者姓（或全体作者）
- 出版年份
- 论文标题（或关键词）
- 期刊/来源（AER/JPE/QJE/NBER等）

整理为待核查列表，格式：
```
[REF_ID] Author(year) Title - Journal
```

---

## 步骤 2：逐条 WebSearch 核查

对每条引用执行以下核查流程。

### 搜索策略（按顺序尝试，任一命中即通过）

**搜索1**（精确匹配）：
```
"[第一作者姓]" "[年份]" "[标题关键词前5词]" "[期刊名]"
```

**搜索2**（放宽标题）：
```
[第一作者姓] [年份] [标题主要名词] site:aeaweb.org OR site:nber.org OR site:jstor.org
```

**搜索3**（最宽泛）：
```
[作者] [年份] economics "[期刊名]"
```

### 判断规则

**✅ PASS（通过）**：搜索结果中有明确页面（AEA/NBER/JSTOR/Springer/SSRN）显示：
- 作者姓名与原引用一致（允许中间名缺失）
- 年份误差 ≤ 1年
- 标题关键词有3个以上重合

**⚠ WARN（警告，需人工确认）**：
- 找到同作者同年份但标题略有出入（可能是引用格式差异）
- 找到相似论文但发表在不同期刊
- 搜索结果质量低（仅来自引用数据库，无原文页面）

**❌ FAIL（失败，很可能是假引用）**：
- 3次搜索均未找到与原引用匹配的页面
- 找到该作者该年份论文，但标题完全不同
- 期刊+作者+年份全部不匹配

---

## 注意事项

- 每条引用至少执行**2次不同角度搜索**，避免单次搜索遗漏
- 中文论文：用中文关键词搜索 CNKI/知网，标准同上
- NBER Working Paper：找到工作论文页面即视为 PASS，即使尚未正式发表
- 搜索过程中若发现更好的同主题真实引用，记录在备注栏
- **不要修改 ECO_LIT.md 本身**——只输出核查报告，由用户决定如何处置
