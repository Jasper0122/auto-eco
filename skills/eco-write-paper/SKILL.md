# Eco-Write-Paper: 国内/国际投稿版计量经济学论文写作技能（完整独立版）

## 你的任务
读取 ECO_LIT.md、ECO_DATA_PROFILE.md、ECO_BACKGROUND.md、ECO_REFS_VERIFIED.md、ECO_CONTRIBUTION.md、ECO_RESULTS.md，默认按照国内经管类/CSSCI实证论文规范生成完整论文草稿；若用户明确指定 AER/AEJ/QJE 等国际期刊，再切换到国际投稿模式。环境允许时生成 `paper.tex` 和 `paper.pdf`。

## 参数
$ARGUMENTS：研究主题；可选附加模式说明（如“国内期刊”“CSSCI”“AER”“AEJ Applied”“QJE”）

默认模式：`domestic`（国内经管类/CSSCI实证论文）

模式识别：
- 若用户未明确指定期刊或只说“论文/中文论文/国内论文/CSSCI/中文期刊”，使用 `domestic`。
- 若用户明确指定 AER、AEJ、QJE、Econometrica、international、英文投稿，使用 `international`。
- 若用户指定具体中文期刊，仍使用 `domestic`，但按目标期刊微调摘要、字数、参考文献格式和标题层级。

## 国内经管类/CSSCI模式（默认）

本模式是默认成文规则。目标不是把国际论文直译成中文，而是生成符合国内经济学、管理学、数量经济、产业经济、区域经济、公共政策类期刊常见审稿期待的实证论文：政策语境充分、理论机制完整、研究假说清楚、实证结果扎实、政策启示能对应结果。

### 自动成文流程（必须执行）

国内模式不得一次性生成全文。必须采用“计划—分节—质检—循环—整合”的写作管线，借鉴自动科研项目的阶段化逻辑：每一节先有任务、证据和字数目标，写完后独立检查；不达标则回到该节扩写或修订，直到达到门槛或触发停止条件。

核心输出文件：

```text
ECO_WRITING_TODO.md          写作总任务清单、状态、质量门槛
ECO_SECTION_PLAN.md          分节大纲、每节证据、字数目标、表图/文献映射
reference_papers/            可公开获取的目标期刊样本文献与格式观察记录
sections/
  01_intro.md
  02_policy_theory.md
  03_research_design.md
  04_results.md
  05_mechanism.md
  06_heterogeneity.md
  07_extension.md
  08_conclusion_policy.md
ECO_WRITING_QA.md            每节质检记录、缺口、修订轮次
ECO_PAPER_DRAFT.md           通过质检后的整合稿
paper.tex / paper.pdf        LaTeX和PDF
```

执行顺序：

1. **创建 todo list**：写入 `ECO_WRITING_TODO.md`，列出所有章节、目标字数、输入证据、依赖文件、状态。
2. **采集格式样本**：若环境允许联网，优先从期刊官网、开放期刊平台或机构知识库下载1-2篇同类高水平中文期刊公开 PDF，保存到 `reference_papers/`，并写入 `reference_papers/FORMAT_REFERENCE_NOTES.md`。只模仿结构、版式和行文组织，不复制正文内容；不得使用盗版来源。
3. **生成分节计划**：写入 `ECO_SECTION_PLAN.md`，每节必须包含“本节目的、核心论点、可用证据、必引政策/文献/表图、字数下限、质量门槛”，并注明参考样本文献对应的格式启发。
4. **逐节写作**：依次生成 `sections/01_*.md` 到 `sections/08_*.md`。每节只围绕本节目标写，不提前写其他节内容。
5. **逐节质检**：每写完一节，更新 `ECO_WRITING_QA.md`，按下方质量门槛打分。
6. **不达标循环**：任何一节未达标，必须回到该节修订，最多3轮；不得直接进入最终整合。
7. **全文整合**：所有章节通过后，才合并为 `ECO_PAPER_DRAFT.md`。
8. **全文总检**：检查总字数、结构、引用、表图、政策语境、结果一致性；未达标则回到相应章节修订。
9. **生成 LaTeX/PDF**：总检通过后再生成 `paper.tex` 和 `paper.pdf`。

### 写作 todo 模板

`ECO_WRITING_TODO.md` 必须采用如下格式：

```markdown
# 写作任务清单

## 全文目标
- 模式：国内经管类/CSSCI实证论文
- 目标字数：10,000-15,000中文字符；CSSCI/高质量实证论文建议12,000-18,000中文字符
- 最低通过字数：10,000中文字符（只统计正文中文字符，不含摘要、表图、公式、参考文献和附录）
- 目标参考文献：20-35篇
- 目标中文文献：不少于8篇（不足则列待补充清单）

## 章节任务
| 章节 | 文件 | 目标字数 | 输入证据 | 表图 | 状态 | 修订轮次 |
|---|---|---:|---|---|---|---:|
| 一、引言 | sections/01_intro.md | 1800-2400 | 政策语境+文献+贡献 | 无 | pending | 0 |
| 二、政策背景、制度事实与理论分析 | sections/02_policy_theory.md | 1800-2600 | ECO_BACKGROUND+政策文件 | 可选机制图 | pending | 0 |
| 三、研究设计 | sections/03_research_design.md | 1200-1800 | ECO_DATA_PROFILE | 变量表/描述统计 | pending | 0 |
| 四、实证结果分析 | sections/04_results.md | 1800-2600 | ECO_RESULTS+tables | 表1-表4/图1 | pending | 0 |
| 五、机制检验 | sections/05_mechanism.md | 1000-1500 | ECO_RESULTS+ECO_LIT | 机制表 | pending | 0 |
| 六、异质性分析 | sections/06_heterogeneity.md | 1000-1500 | ECO_RESULTS+ECO_LIT | 异质性表 | pending | 0 |
| 七、进一步分析 | sections/07_extension.md | 800-1200 | 多结果/替代解释 | 扩展表 | pending | 0 |
| 八、研究结论与政策启示 | sections/08_conclusion_policy.md | 1000-1500 | 全文结果 | 无 | pending | 0 |
```

状态只能使用：`pending`、`drafted`、`needs_revision`、`passed`。

### 分节计划要求

`ECO_SECTION_PLAN.md` 中每节必须写：

- 本节功能：回答什么问题。
- 核心论点：2-4条，不超过一屏。
- 证据映射：来自哪个 `ECO_*.md`、哪个表、哪个政策来源、哪篇文献。
- 字数下限：低于下限必须扩写。
- 禁止事项：本节不能写什么，避免重复。
- 质检门槛：该节通过条件。

示例：

```markdown
## 4 实证结果分析
- 功能：证明主效应可信且稳健。
- 核心论点：
  1. 基准回归显示政策显著提高Y。
  2. CS-DiD与TWFE方向一致，交错处理偏误有限。
  3. 平行趋势与安慰剂支持识别。
  4. 稳健性检验排除主要规格依赖。
- 证据映射：
  - ECO_RESULTS.md：Table 2、Table 3、Table 4
  - tables/figure1_event_study.png
- 字数下限：2200字符
- 禁止事项：不得讨论政策启示；不得引入未报告的新稳健性。
- 通过条件：每个主系数均有SE/p值/经济意义；每个稳健性说明对应的识别威胁。
```

### 分节质检门槛

每节写完后必须更新 `ECO_WRITING_QA.md`。每节满分10分，低于8分必须修订。

通用评分：
- 结构完整（2分）：是否完成本节应有小节。
- 证据充分（2分）：是否引用对应表、政策、文献或数据。
- 主题聚焦（1.5分）：是否围绕本节功能，不跑题。
- 字数达标（1分）：是否达到本节最低字数。
- 结果准确（1.5分）：系数、年份、样本、政策事实是否与状态文件一致。
- 国内行文（1分）：是否符合国内论文语气和逻辑。
- 不重复不注水（1分）：是否避免同义反复和空泛政策话。

`ECO_WRITING_QA.md` 格式：

```markdown
# 写作质检记录

| 章节 | 轮次 | 字数 | 得分 | 结论 | 主要问题 | 下一步 |
|---|---:|---:|---:|---|---|---|
| 1 引言 | 1 | 1800 | 7.5 | needs_revision | 文献缺口不足；贡献不具体 | 扩写文献述评和贡献 |
| 1 引言 | 2 | 2500 | 8.5 | passed | - | 进入下一节 |
```

### 循环修订规则

- 每节最多修订3轮。
- 若第3轮仍低于8分，允许继续整合，但必须在 `ECO_WRITING_QA.md` 标记 `residual risk`，并在最终汇报提示用户。
- 若正文中文字符低于10,000字，不得生成最终版、`paper.tex` 或 `paper.pdf`；必须优先扩写低于目标的章节。
- 若参考文献少于20篇或中文文献少于8篇，不得编造文献；必须生成“待补充文献检索清单”，并继续完成正文。
- 若政策语境缺权威来源，写“待核查”，并在 todo 中标记为阻塞项或风险项。

### 最终整合规则

只有当以下条件满足时，才能写 `ECO_PAPER_DRAFT.md`：

- 所有章节状态为 `passed`，或仅有不超过2个 `residual risk`。
- 全文正文不含摘要、表图、公式、参考文献和附录不少于10,000中文字符；低于10,000字不得标记为最终版。
- 所有核心系数与 `ECO_RESULTS.md` 一致。
- 所有政策事实来自 `ECO_BACKGROUND.md` 或权威来源；未核实处明确标注。
- `ECO_REFS_VERIFIED.md` 中 FAIL 引用已删除。

```text

  （二）理论机制
  （三）研究假说

  （一）模型设定
  （二）变量说明
  （三）数据来源与描述统计
四、实证结果分析
  （一）基准回归
  （二）识别有效性/内生性处理
  （三）稳健性检验

七、进一步分析/拓展检验
八、研究结论与政策启示

```

篇幅控制：
- 普通中文经管论文：8,000-12,000中文字符。
- 本技能国内模式硬性最低：10,000中文正文字符，不含摘要、表图、公式、参考文献和附录。
- CSSCI/高质量实证论文：建议12,000-18,000中文字符；若用户未特别要求长文，默认先保证10,000-15,000中文正文字符。
- 摘要：200-300字，第三人称，包含研究目的、方法、结果、结论。
- 关键词：3-5个。
- 参考文献：20-35篇；中文文献不少于8篇，除非研究主题高度国际化；近5年文献建议占1/3以上。
- 表图：主文5-8个核心表图；补充稳健性、变量字典和扩展结果放附录。

标题层级：
- 国内模式默认使用中文期刊常见标题：一级标题为“一、二、三”，二级标题为“（一）（二）（三）”；若目标期刊明确使用 `1`、`1.1`、`1.1.1`，再按目标期刊调整。
- 一级标题单独成行；二级标题服务于逻辑推进，避免过碎。

### 通用政策语境识别

国内论文引言和政策背景通常需要把研究问题嵌入国家战略、重大政策、重要会议或政府文件，但必须通用匹配，不能把某一主题的政策材料写死。

先根据研究主题和核心解释变量识别政策域：

| 政策域 | 典型关键词 | 可检索政策语境 |
|---|---|---|
| 数字经济/信息基础设施 | 数字经济、宽带、互联网、数据、平台、AI、工业互联网 | 数字中国、网络强国、数实融合、制造业数字化转型、数字基础设施 |
| 绿色低碳/环境治理 | 碳排放、污染、绿色、能源、环保、生态 | 双碳目标、生态文明建设、绿色发展、新发展理念、美丽中国 |
| 制造业/产业升级 | 制造业、产业链、技术改造、设备更新、TFP | 制造强国、新型工业化、现代化产业体系、新质生产力、产业基础再造 |
| 区域发展/县域经济 | 区域、城市、县域、乡村、中西部、园区 | 区域协调发展、乡村振兴、共同富裕、西部大开发、长江经济带、黄河流域 |
| 创新创业 | 专利、研发、创新、创业、科技 | 创新驱动发展、科技自立自强、国家创新体系、专精特新 |
| 金融财政 | 信贷、融资、财政、税收、金融科技 | 金融服务实体经济、普惠金融、防范化解金融风险、财政高质量发展 |
| 开放贸易 | 出口、外资、自贸区、关税、贸易 | 高水平对外开放、双循环、一带一路、自贸区/自贸港 |
| 劳动民生 | 就业、收入、教育、健康、人口、社保 | 共同富裕、就业优先、教育强国、健康中国、人口高质量发展 |
| 企业治理/营商环境 | 国企、民企、治理、营商环境、统一市场 | 国企改革、民营经济促进、优化营商环境、全国统一大市场 |

政策语境写作流程：
1. 从主题中提取政策域，不确定时选择最贴近核心解释变量X的政策域。
2. 优先使用 `ECO_BACKGROUND.md` 已核实的政策材料。
3. 若材料不足，使用 WebSearch 检索权威来源：中国政府网、新华社、人民网、国家统计局、国务院部门官网、省市政府官网。
4. 写作时采用“国家战略背景 -> 现实问题 -> 本文研究问题”的三段递进。
5. 不得凭记忆编造“习近平指出”“中央强调”等表述；需要引用讲话时必须有来源，并以转述为主。

政策语境不是宣传稿。每个政策段落必须服务一个学术目的：
- 说明研究问题的重要性。
- 说明政策冲击或制度环境。
- 解释理论机制。
- 引出政策启示。

### 国内引言写法

国内模式引言建议写7段：

1. **国家战略背景**：用权威政策文件或会议精神说明主题为何重要。不要超过2段。
2. **现实问题与矛盾**：给出宏观事实、行业痛点或政策难题。
3. **学术问题**：从现实问题转为可识别的经济学问题。
4. **文献述评**：概括已有文献的主线和不足，不罗列。
5. **本文设计**：说明数据、政策冲击/识别策略和核心模型。
6. **主要发现**：用数字汇报主结果、机制和异质性。
7. **边际贡献与结构安排**：贡献通常写3点，随后说明章节安排。

贡献写法必须具体：
- 理论贡献：补充了哪条机制或哪类理论解释。
- 识别贡献：使用了什么更干净的政策冲击、工具变量或准自然实验。
- 数据贡献：使用了更细粒度、更长时期或更高质量数据。
- 政策贡献：对哪类政策优化有启示。

### 政策背景、理论机制与假说

第2节是国内论文的核心风格差异，不能省略。

`2.1 政策背景与制度事实`
- 写政策演进、政策目标、实施机制、处理对象、批次/时点、政策强度。
- 说明为什么该制度事实能服务识别设计。
- 每个关键事实必须有来源；没有来源写“待核查”，不得编造。

`2.2 理论机制`
- 至少写2条机制，最多4条。
- 每条机制采用“理论逻辑 -> 可观测变量 -> 预期方向”的结构。
- 对机制变量不可得的机制，可以写为理论解释，但不能伪装成已检验机制。

`2.3 研究假说`
- H1：主效应。
- H2/H3：机制假说。
- H4：异质性假说（如地区、规模、所有制、资源禀赋、市场化程度）。
- 每个假说前必须有理论推导，不得只列假说。

### 研究设计

第3节按国内期刊习惯展开：

`3.1 模型设定`
- 给出基准模型、识别模型和必要的机制/异质性模型。
- 解释变量、固定效应、控制变量和误差项。
- 说明标准误聚类层级。

`3.2 变量说明`
- 被解释变量、核心解释变量、控制变量、机制变量、异质性变量。
- 用表格列出变量名称、符号、定义、数据来源。

`3.3 数据来源与描述统计`
- 写数据来源、样本期间、样本筛选、缺失和极端值处理。
- 描述统计表必须用三线表。

### 实证结果组织

`4 实证结果分析`
- 基准回归：递进规格，解释系数大小和经济意义。
- 识别有效性/内生性：平行趋势、安慰剂、IV第一阶段、断点操纵等按方法选择。
- 稳健性：替换变量、缩尾、样本调整、固定效应调整、聚类层级调整。

`5 机制检验`
- 每个机制先回到理论，再汇报结果。
- 中介效应/机制回归使用保守表述：“支持该机制”“与机制假说一致”，不轻易写“证明”。

`6 异质性分析`
- 分组必须来自理论，不机械遍历。
- 必须报告组间差异或交互项检验，不能只比较显著性。

`7 进一步分析/拓展检验`
- 用于多结果变量、替代解释排除、空间溢出、政策强度、长期效应、福利含义等。
- 若结果不足，宁可省略第7节，不强行凑。

`8 研究结论与政策启示`
- 先写3-4条研究结论，每条对应实证发现。
- 再写3-4条政策启示，每条必须对应前文结果。
- 避免泛泛写“加强顶层设计”“完善配套政策”；必须说明针对谁、通过什么工具、解决什么约束。
- 最后一段写研究不足与未来展望。

### A. AER引言8段结构（详细规范）

基于AER 2020-2021年16篇论文提炼的标准模板，每段均有明确功能和写作要求。

---

**【第1段：宏观钩子】**

**功能：** 从现实政策问题或经济现象切入，建立重要性。

**写作要求：**
- 用1-2句描述现实问题的规模（直观统计数字，如：截至[年份]，中国有X亿人口从事Y行业）
- 说明这一问题的理论重要性与现实政策关切
- 避免空洞开头，第一句必须包含具体数据

**示例句式：**
> 在过去三十年里，中国城镇化率从[X]%上升至[Y]%，超过[Z]亿农村人口进入城市。这一规模空前的人口流动深刻改变了劳动力市场结构，然而其对[研究问题]的因果影响至今仍不清晰。

---

**【第2段：文献缺口】**

**功能：** 综述现有研究的共同局限，明确因果识别的缺失。

**写作要求：**
- 承认现有文献的贡献（"大量研究表明..."）
- 点明共同局限（内生性来源：选择偏误/反向因果/遗漏变量）
- 用一句明确的话收尾："然而，[研究问题]至今缺乏严格的因果证据"

**示例句式：**
> 现有文献普遍发现[X与Y的相关性]（Author1, Year; Author2, Year）。然而，这些研究面临共同的内生性挑战：[内生性来源，如：政策实施并非随机，往往针对经济条件较差的地区]。因此，[研究问题]的因果效应至今尚无定论。

---

**【第3段：本文贡献（含核心数字，必须具体）】**

**功能：** 一句话概括本文，立即给出核心数字建立量级感。

**写作要求（严格）：**
- 第一句说明识别策略来源
- **第二句必须给出具体数字，格式为：**
  > β = X（SE = Y，p < Z），相当于样本均值的 [β/mean × 100]%
- 第三句说明结论稳健性

**标准格式（直接套用）：**
> 本文利用[外生冲击来源]，识别[研究问题]的因果效应。我们的核心发现是：[X变量]每增加1[单位]，[Y变量]增加/降低[β]（SE = [se]，p < [p]），相当于样本均值（[mean]）的[β/mean×100]%。这一效应在[稳健性检验类型]下保持稳健。

**数字规范（来自AER第十五章）：**
- β必须报告4位有效数字（如：0.2341，而非"约0.2"）
- SE在括号内紧随β
- 效应量级判断：
  - β/mean < 5%：经济意义较小，需在正文专门讨论其可解释性
  - β/mean 在 5%-20%：经济意义中等，AER常见范围
  - β/mean > 20%：经济意义较大，需谨慎，说明原因
- **Back-of-Envelope换算（必须做）：** 将系数换算为更直观单位，例如：
  - "多接受1年教育 → 工资增加X% → 终生收入多Y万元"
  - "政策覆盖N个城市 → 总计带动Z亿元投资"

---

**【第4段：识别策略细节】**

**功能：** 说明外生变动来源及其准随机性，并点明潜在威胁。

**写作要求：**
- 描述数据特点（时间、地区、样本量）
- 明确外生变动的来源及为何外生
- 说明平行趋势/工具变量有效性的验证逻辑
- 点明潜在威胁（同期政策/选择偏误/溢出效应）及本文的应对方案

**示例句式：**
> 本文使用[数据集]，覆盖[N]个[个体]、[T]个时期，共[N_obs]个观测。识别变异来源于[外生冲击]，[冲击]的实施[时间]由[行政/历史/自然原因]决定，与地方经济条件无关。我们通过[事件研究图/平衡性检验]验证了[准随机性]。本文最主要的内生性威胁是[威胁1]，我们通过[应对方案]加以处理。

---

**【第5段：主要结果1（基准回归）】**

**功能：** 汇报基准回归核心数字。

**写作要求：**
- 报告主系数（β、SE、显著性水平）
- 说明固定效应规格
- 报告样本量
- 换算经济意义（必须给百分比变化）

**示例句式：**
> 基准回归结果显示，[X]每增加1[单位]，[Y]显著[上升/下降][β]（SE = [se]），在[1%/5%/10%]水平上显著。在包含[个体FE + 年份FE]、将标准误聚类至[聚类层级]的主规格下（N = [样本量]），该效应相当于[Y]样本均值的[%]。

---

**【第6段：主要结果2（机制与异质性）】**

**功能：** 汇报机制分析和异质性分析的核心结论。

**写作要求：**
- 机制分析：[X]如何通过[中间变量]传导至[Y]，传导比例多少
- 异质性分析：哪类群体效应更强/更弱，与理论是否一致

**示例句式：**
> 机制分析表明，[X]主要通过[中间变量M]渠道影响[Y]：[X]使[M]显著[变化][γ]（SE = [se]），在控制[M]后，[X]对[Y]的系数从[β₀]降至[β₁]，下降约[Δ%]，说明[M]传导了约[Δ/β₀×100]%的总效应。异质性分析发现，效应在[子群体]中更为显著（[β_h] vs [β_l]，差异显著，p = [p]），这与[理论机制]的预测一致。

---

**【第7段：文献贡献（Literature）】**

**功能：** 明确本文对3-5个文献流的增量贡献。

**标准格式（直接套用）：**

```
本文与以下文献密切相关：

第一，[文献领域1]。[该领域代表性发现，2-3句]。本文的贡献在于[与现有文献相比的增量，如：
首次提供了X情境下的因果证据 / 利用更外生的识别策略 / 将研究对象拓展至Y群体]。

从而[贡献点]。

第三，[文献领域3]。[2-3句]。本文通过[机制分析/异质性分析]表明[贡献]。

（可选第四、第五个文献流，格式同上）
```

**注意：** 每个文献流至少2句引用+1句贡献声明，避免仅罗列名字而不说明增量贡献。

**【第8段：论文结构（Roadmap）】**

**功能：** 说明后续章节安排（必须有，AER强制要求）。

**标准句式：**

### C. 计量模型方程规范（DiD四种变体）

**重要：** 根据 ECO_RESULTS.md 中的识别策略，选择对应变体写方程。

---

**【变体1：队列强度DID（Cohort × Intensity，Duflo型）】**

**适用场景：** 利用"政策强度跨地区变异 × 受影响队列跨群体变异"的双重交叉识别。代表：Chen等(2020) 上山下乡运动研究。

**标准方程：**
$$Y_{igc} = \alpha_c + \mu_{gp} + \beta(\text{Intensity}_c \times \text{Treated}_g) + (\Lambda_c \times \mu_g) + \mathbf{X}_{igc}'\delta + \varepsilon_{igc} \quad (1)$$

- $\text{Intensity}_c$：县/地区级政策强度（如：知青密度、覆盖率）
- $\text{Treated}_g$：受影响的出生年份/队列虚拟变量
- $\Lambda_c \times \mu_g$：县基础水平 × 队列固定效应（关键控制项，防止时变混淆）
- $\alpha_c$：县固定效应；$\mu_{gp}$：队列×省份固定效应

**配套稳健性（必做）：**
- 伪处理队列安慰剂（控制组一分为二，检验前期无效应）
- 城市样本安慰剂（不受政策影响的对照群体）
- 合成控制法（不依赖平行趋势假设的备选识别）

---

**【变体2：单一冲击事件研究（Single Event DiD）】**

**适用场景：** 单一政策事件，处理时间统一，多个控制单元。代表：Deryugina等(2020) 卡特里娜飓风研究。

**标准方程（事件研究规格）：**
$$Y_{it} = \alpha_i + \gamma_t + \sum_{k \neq -1} \beta_k \cdot (D_i \times \mathbf{1}[t - t_0 = k]) + \mathbf{Z}_{it}'\delta + \varepsilon_{it} \quad (1)$$

- $k = -K, \ldots, -1, 0, 1, \ldots, L$，以 $k = -1$ 为基准期（归一化为0）
- $\beta_k (k < 0)$ 联合不显著 → 平行趋势成立
- $\beta_k (k \geq 0)$ 显著 → 政策效应存在

**配套要求：**
- 用多个历史窗口验证长期平行趋势
- 单一处理单元时加合成控制法 + 排列推断（Permutation Inference）

---

**【变体3：交错DiD + IV双策略】**

**适用场景：** 第一步用准随机DiD解决选择偏误，第二步用IV处理测量误差或依从性问题。代表：Fetzer等(2021) 阿富汗安全移交研究。

**第一阶段（DiD识别外生变动）：**
$$X_{it} = \pi \cdot Z_{it} + \phi' \mathbf{W}_{it} + \mu_i + \lambda_t + \nu_{it} \quad (2)$$

**第二阶段（IV估计因果效应）：**
$$Y_{it} = \beta \cdot \hat{X}_{it} + \delta' \mathbf{W}_{it} + \mu_i + \lambda_t + \varepsilon_{it} \quad (3)$$

**诊断要求：**
- 第一阶段 F 统计量（优先报告 Kleibergen-Paap rk Wald F，聚类SE下适用）
  - F > 10：满足相关性（Stock-Yogo经验阈值）
  - F > 104.7：满足Montiel Olea-Pflueger最严格检验（AER近年趋势）
- 可加入地区特异性线性时间趋势控制基线差异

---

**【变体4：梯度DID + 三重差分（Triple-DiD）】**

**适用场景：** 测量"斜率变化"而非"水平变化"，第三维度为连续强度变量。代表：Truglia(2020) 收入透明度研究。

**标准三重差分方程：**
$$Y_{ict} = \beta_1 (\text{Treat}_i \times \text{Post}_t) + \beta_2 (\text{Treat}_i \times \text{Post}_t \times \text{Intensity}_c) + \gamma_{ic} + \delta_{it} + \lambda_{ct} + \mathbf{X}_{ict}'\rho + \varepsilon_{ict} \quad (1)$$

- $\gamma_{ic}$：个体×强度固定效应
- $\delta_{it}$：个体×时间固定效应
- $\lambda_{ct}$：强度组×时间固定效应
- $\beta_2$：核心参数，捕捉梯度在政策前后的变化（DDD估计量）

**配套稳健性：**
- 跨国安慰剂：对无类似政策变化的对照国家重复检验
- 用预测值（如网络接入率）作为第三维度，而非直接观测值

---

**【标准DiD / 面板FE（基础规格）】**

当 ECO_RESULTS.md 中识别策略为标准双向固定效应DiD时，使用此基础方程：

$$Y_{it} = \alpha_i + \gamma_t + \beta(\text{Treat}_i \times \text{Post}_t) + \mathbf{Z}_{it}'\delta + \varepsilon_{it} \quad (1)$$

- $\alpha_i$：个体固定效应（吸收时不变差异）
- $\gamma_t$：时间固定效应（吸收共同时间趋势）
- $\beta$：核心DiD估计量，捕捉政策的平均处理效应（ATT）
- 标准误在[聚类层级]层面聚类（通常为处理单元层级）

**IV策略（补充）：**
$$X_{it} = \pi Z_{it} + \mathbf{W}_{it}'\theta + \mu_i + \lambda_t + \nu_{it} \quad (2)$$
$$Y_{it} = \beta \hat{X}_{it} + \mathbf{W}_{it}'\delta + \mu_i + \lambda_t + \varepsilon_{it} \quad (3)$$

### E. 数字规范（AER第十五章）

**核心β报告规范：**
- β必须4位有效数字（例：0.2341，不写"约0.23"）
- SE在括号内紧随β，同样4位有效数字
- 显著性以p值或星号表示（推荐同时报告）

**经济意义量级判断（必须换算并报告）：**
- 效应/对照组均值 **< 5%**：经济意义较小，需在正文专门讨论（"虽然统计上显著，但经济意义有限，原因可能在于..."）
- 效应/对照组均值 **5%-20%**：经济意义中等，AER常见区间，直接报告即可
- 效应/对照组均值 **> 20%**：经济意义较大，需谨慎，解释驱动因素（"这一较大效应可能源于..."）

**Back-of-Envelope计算（引言第3段必须包含）：**
将回归系数换算为更直观的经济量级，格式示例：
- "以样本平均[X]值（=[mean]）计算，效应相当于每年额外[直观量]"
- "若将这一效应外推至全国[N]个[个体]，相当于总计影响[Z]亿元/[Z]万人"
- "多接受1年教育 → 工资增加X% → 以30年工作期计，终生收入增加Y万元"

**样本描述必须包含：** N个观测，X个个体，T个时期（三个数字缺一不可）

### F. 稳健性检验写作模板（对应eco-analysis 5项稳健性）

基于AER第五章稳健性清单，对 eco-analysis 生成的5项标准稳健性检验，用以下句式逐一描述。

**写作格式（表4汇报，正文逐一解读）：**

> 表4汇报了多项稳健性检验，以验证基准结论的可靠性。

**1. 替换被解释变量（Table 4，列1）：**
> 列(1)使用[Y的替代测量方式，如：取对数/二值化/替代指标]替换基准被解释变量。[X]的系数为[β]（SE = [se]，p = [p]），与基准结果[高度一致/方向一致且仍显著]，说明结论并非特定变量构造方式所致。

**2. 缩尾处理（Table 4，列2）：**
> 列(2)对被解释变量和核心解释变量分别在1%和99%分位数处进行缩尾处理（Winsorize），以排除极端值干扰。处理后系数为[β]（SE = [se]），与基准[β₀]相差[Δ%]，结论保持稳健，表明基准结果并非由极端观测值驱动。

**3. 更换聚类层级（Table 4，列3）：**
> 列(3)将标准误聚类层级从[原层级，如：县级]提升至[更粗层级，如：省级]，以处理可能存在的更高层面序列相关。在更保守的推断标准下，核心系数的t统计量仍达到[显著水平]（p = [p]），识别不受聚类设定影响。

**4. 剔除极端/特殊样本（Table 4，列4）：**
> 列(4)剔除[特殊样本，如：直辖市/政策实施前已有类似政策的省份/样本量过小的组别]，剩余样本核心系数为[β]（SE = [se]），与基准结论[一致]，排除了这些特殊观测值对结果的影响。

**5. 不含控制变量（Table 4，列5）：**
> 列(5)仅保留固定效应，不加入任何控制变量，以验证结论并非依赖于特定控制变量的选择。系数为[β]（SE = [se]），与含控制变量的基准规格（β₀ = [β₀]）高度接近（差异[Δ%]），证明结论由处理效应本身驱动，而非控制变量构造。

> 综合以上[N]项稳健性检验，基准回归的核心结论对变量测量方式、极端值处理、标准误设定及样本选择均保持稳健。

## 可投稿论文格式校准

本技能生成的是“投稿初稿”，不是演示性报告。默认目标为国内经管类/CSSCI实证论文；只有用户明确指定国际期刊时，才切换到 AEA/AER、AEJ: Applied Economics、QJE、Econometric Society 的国际审稿格式。

### 调研依据（写入规范时已吸收）

- 国内经管类期刊常见要求：中文摘要约200-300字，关键词3-5个，正文常用“一、二、三”和“（一）（二）（三）”标题层级，图表使用统一字号三线表和规范资料来源，参考文献遵循GB/T 7714-2015并使用 `[1]` 编号；高质量实证论文通常需要较完整的政策背景、理论机制、研究假说、稳健性、机制和异质性分析。
- AER/AEA：投稿 PDF；首页放标题、作者和单位；摘要不超过100词；AER 平均文章约35-36个排版页，建议不超过40页（11pt、1.5倍行距、1英寸边距）或45页（12pt、1.5倍行距、1英寸边距）；实证论文需提供可复现的数据、代码和计算细节；主文必须自洽，核心数据来源、变量、方法不能只放补充附录。
- AEJ: Applied Economics：摘要不超过100词；建议40页/45页上限；主文可含 in-paper appendix，补充附录只能放非核心扩展和稳健性，主文必须自洽。
- QJE：稿件顺序为 title page、abstract、text、affiliations、appendix、references、notes、tables、figures；摘要不超过250词；表格和图形各自编号，表格用罗马数字，表格应有标题和必要脚注，避免竖线、彩色文字和阴影。
- Econometric Society：页码、摘要、关键词和作者信息置于首页；正文组织要最大化可读性；只给后文引用的方程编号；方程、表格、图和命题等必须清楚编号并可交叉引用。

## 论文结构与写作内容（国际投稿版，仅用户明确指定时使用）

国际模式正文采用以下 8 节主结构。国内模式不得套用本节，而应使用上文“国内经管类/CSSCI模式（默认）”的结构。

### 首页：标题、作者信息、摘要、关键词、JEL、披露

**标题**
- 格式：`[因果问题]：来自[政策/自然实验/制度冲击]的证据`
- 标题不超过28个汉字或18个英文词；必须包含研究对象和识别来源，避免“研究”“分析”“探讨”等空泛词。

**作者与单位**
- 中文稿保留“作者预留、单位预留、邮箱预留”占位。
- 若生成投稿匿名版，删除作者、单位、致谢中可识别身份的信息。

**摘要**
- 中文期刊版：250-350字。
- AEA版：另附英文摘要，不超过100词。
- QJE版：英文摘要不超过250词。
- 摘要必须四句完成：研究问题；识别设计和数据；2-3个核心结果；贡献或政策含义。
- 摘要禁止引用文献、禁止脚注、禁止“本文可能具有重要意义”这类空话。
- 摘要至少报告一个核心系数或效应量，但不能塞满全部稳健性细节。

**关键词与 JEL**
- 关键词5个以内，按“政策/冲击、结果变量、方法、场景、机制”排序。
- JEL 3-5个，优先从 `ECO_LIT.md` 和研究主题推断；不确定时写“JEL: 待补充”。

**首页脚注/披露**
- 写入占位：资金来源、利益冲突、IRB/RCT（如适用）、AI辅助说明、数据和代码可得性。
- AEA要求 AI 不能列为作者；如使用 AI 辅助写作，需由作者负责事实核查并在提交流程中说明。

### 一、引言（8段，1800-2500字）

引言仍采用上文【规范A】8段结构，但必须按投稿版压实：

1. **问题重要性**：第一段给出真实世界规模、政策成本或福利含义，必须有具体数字和来源线索。
2. **识别难题**：说明为什么普通相关性、OLS或宏观趋势图不能回答因果问题。
3. **本文做什么**：一句话说清“利用什么冲击、识别什么效应、用什么数据”。
4. **核心结果**：报告主系数、标准误、p值或置信区间、效应占均值比例。
5. **识别可信度**：平行趋势/第一阶段/安慰剂/排除同期政策，集中写最关键的2-3个验证。
6. **机制和异质性**：只报告最有理论支撑的机制和最关键异质性，不堆全部结果。
7. **贡献定位**：对3条文献流分别写“已有文献做了什么、还缺什么、本文增加什么”。
8. **结构安排**：用一句路线图收尾。

质量门槛：
- 引言第3-5段必须让审稿人不用翻表也能知道识别设计、样本、主结果和可信度。
- 文献贡献不能写“丰富了”“补充了”就结束，必须具体到数据层级、识别策略、机制、场景或外部有效性中的至少两项。

---

### 二、制度背景与识别环境（1000-1500字）

这一节优先读取 `ECO_BACKGROUND.md`，每个关键事实必须有 URL 来源或正式文件来源。不是泛泛介绍政策，而是服务识别。

**2.1 政策/冲击的制度细节**
- 政策发布、实施、批次、处理组名单、覆盖范围、执行主体。
- 明确处理时间 `t0` 和处理单元层级。
- 写清哪些单位未处理、为何能作为对照组。

**2.2 处理分配机制**
- 说明入选/暴露/强度变化由什么规则决定。
- 讨论处理分配是否与预期结果变量相关。
- 若入选并非随机，必须提出可检验的选择威胁，并在后文用平衡性、趋势、控制、匹配或安慰剂回应。

**2.3 可识别变异**
- 说明估计依赖的是跨地区、跨时间、跨队列、强度差异还是工具变量变异。
- 对 staggered adoption，写明不同批次的数量和年份。
- 对连续强度处理，解释单位变化的经济含义。

**2.4 潜在威胁清单**
- 同期政策、预期效应、政策溢出、选择进入、测量误差、样本选择。
- 每个威胁后面写“本文如何处理”，并指向表/图/附录。

---

### 三、数据、样本与变量（1200-1800字）

这一节必须可复现，不能只写“本文使用某数据库”。优先读取 `ECO_DATA_PROFILE.md` 和 `tables/table1_descriptive.*`。

**3.1 数据来源与合并过程**
- 列出每个数据源、提供机构、时间范围、地理/行业覆盖、访问方式。
- 写清样本构造：原始样本量、删除规则、缺失处理、合并键、最终样本。
- 必须报告：观测数 N、个体数、时期数、处理组数、对照组数。

**3.2 变量定义**
- 被解释变量：定义、单位、是否取对数、构造公式、极端值处理。
- 核心解释变量：处理变量/强度/工具变量的构造，处理开始时间如何编码。
- 控制变量：按类别说明经济含义，不逐个机械罗列。
- 机制变量和异质性变量：只定义后文主文会用到的变量，其余放附录。

**3.3 描述统计和平衡性**
- 正文引用 Table I：均值、标准差、处理/对照基线差异、联合平衡性检验。
- 对主结果变量给出直观量级，方便后文换算经济意义。
- 若基线不平衡，不能隐藏；必须说明后文如何处理。

**3.4 数据限制**
- 如实写数据频率、测量误差、不可观察变量、外部有效性限制。
- 不要把数据限制全部放到结论；关键限制应在这里提前交代。

---

### 四、实证策略（1200-1800字）

这一节只讲识别和模型，不汇报结果。根据 `ECO_RESULTS.md` 选择对应模型，方程只编号后文会引用的。

**4.1 主模型**
- 写入主方程，解释每个符号、固定效应、控制变量和聚类层级。
- 明确估计对象：ATE、ATT、LATE、动态效应或强度弹性。
- 对处理变量单位做经济解释。

**4.2 识别假设**
- DiD：平行趋势、无提前反应、SUTVA/无干扰、稳定构成。
- Staggered DiD：说明为什么不用传统 TWFE 作为唯一主结果；正文优先报告 CS-DiD / Sun-Abraham / event-study robust estimator，TWFE作为对照或附录。
- IV：相关性、排他性、单调性、第一阶段强度、LATE解释。
- RDD：连续性、操纵检验、带宽选择。

**4.3 推断**
- 标准误聚类在处理分配层级；聚类数少于50时说明 wild cluster bootstrap 或随机化推断。
- 多重检验使用 BH-FDR 或 Family-wise 纠正时说明 family 定义。

**4.4 预设检验计划**
- 平行趋势/第一阶段。
- 安慰剂和伪处理。
- 稳健性。
- 机制和异质性仅作为理论检验或支持性证据，不得被写成重新选择规格后的主证据。

---

### 五、主结果（1000-1400字）

这一节只承载最重要的因果结果，避免把所有稳健性混进来。

**5.1 基准估计**
- 引用 Table II。
- 逐列解释应简洁：从无控制到主规格的变化，不逐字复述每列。
- 主规格必须报告：系数、标准误、p值或置信区间、样本量、固定效应、聚类层级、控制组均值。

**5.2 经济意义**
- 把主系数换算成占均值比例、弹性或现实单位。
- Back-of-Envelope 只做一个最可信的换算，写出假设边界。
- 若效应很小、很大或不显著，必须解释，不得硬说“显著正向”。

**5.3 动态效应或剂量反应**
- DiD论文引用 Figure I 或 Table III 的事件研究。
- 报告政策前联合检验 p 值和政策后动态路径。
- 对连续强度处理，报告非线性或分位响应（如有）。

---

### 六、识别有效性与稳健性（1200-1800字）

这一节是审稿核心，必须围绕威胁逐项组织，而不是机械列“稳健性1-5”。

**6.1 平行趋势/第一阶段/操纵检验**
- DiD：事件研究图、前导项联合检验、基线平衡。
- IV：第一阶段系数、Kleibergen-Paap F、排他性叙事和安慰剂。
- RDD：McCrary/密度检验、协变量连续性。

**6.2 安慰剂检验**
- 时间安慰剂、空间安慰剂、未受影响结果变量、未受影响样本。
- 每个安慰剂必须说明“如果该威胁存在，应该看到什么；实际看到什么”。

**6.3 稳健性检验**
- 替代变量定义、缩尾、控制变量、固定效应、样本窗口、聚类层级。
- 只在主文放最有说服力的3-5项，其余放补充附录。
- 对每项报告方向、量级是否稳定，不能只说“结论稳健”。

**6.4 溢出和外部干扰**
- 如果存在空间/网络溢出，说明估计量解释会如何变化。
- 若无法排除，写成限制而不是强行声称已解决。

---

### 七、机制与异质性（1000-1600字）

机制和异质性必须理论先行，结果随后；不得机械遍历变量找显著。

**7.1 机制框架**
- 用一段话说明 X 影响 Y 的2-3条理论路径。
- 每条路径对应一个可观测机制变量或中间结果。

**7.2 机制证据**
- 引用 Table V。
- 机制写作采用保守表述：“结果与该渠道一致”“提供支持性证据”。
- 传导比例必须注明是描述性分解，不是严格中介因果效应，除非设计能识别中介因果。

**7.3 异质性**
- 引用 Table VI。
- 只选择理论预先指定的2-3个维度。
- 报告组间差异检验 p 值，不只比较两个系数谁大。
- 如果异质性不显著，如实写“不支持该异质性假设”。

---

### 八、结论（600-900字）

按【规范B】结论结构写作，但投稿版要求更克制：

1. 一句话概括研究问题、识别来源和贡献。
2. 用2-3句总结主结果、识别有效性和机制/异质性，带关键数字。
3. 写政策含义，必须限定适用情境。
4. 写局限性：外部有效性、数据测量、长期效应、未完全排除的威胁。
5. 回扣引言的大问题，不引入新结果、新文献或新数据。

## 表图、附录与参考文献规范

### 主文表图配置

主文最多放6个核心 exhibits：
- Table I：描述统计与基线平衡。
- Table II：主结果/基准回归。
- Figure I：事件研究、第一阶段图或核心动态图。
- Table III：识别有效性（CS-DiD/IV/RDD/安慰剂）。
- Table IV：稳健性。
- Table V：机制或异质性。若两者都重要，可 Table V 机制、Table VI 异质性，但总数不宜超过6。

附录放：
- 额外稳健性。
- 替代样本和替代变量定义。
- 机制和异质性扩展。
- 数据构造细节、变量字典、额外图表。

### 表格格式

- 主文表格编号使用 `Table I`, `Table II`；中文正文可写“表I”。
- 表题要具体，例如 `Table II. Effects of Broadband China on County Manufacturing TFP`，不要写“Regression Results”。
- 表注必须包含：因变量定义、处理变量定义、固定效应、控制变量、聚类层级、样本、显著性星、数据来源。
- 表格不得直接粘贴软件输出；禁止竖线、彩色文字、阴影和过密列。
- 附录表编号为 `Table A.1`, `Table A.2`。

### 图形格式

- 图编号使用 `Figure I`, `Figure II`。
- 事件研究图必须包含点估计、95%置信区间、政策时点竖线、基准期说明、政策前联合检验 p 值。
- 图注应能独立解释图，不依赖正文。
- 图中文字不能过小；若生成 LaTeX/PDF，优先使用 PDF/EPS/高分辨率 PNG。

### 参考文献

- 只列正文和附录实际引用的文献。
- 优先使用作者-年份制，中文稿也保留英文文献原题。
- 对工作论文、政策文件、数据集给出稳定链接、机构、年份和访问日期。
- 若 `ECO_REFS_VERIFIED.md` 有 FAIL 引用，禁止写入参考文献。

### 复现与披露

论文末尾必须有以下占位小节：

```markdown
## Data and Code Availability
[说明数据来源、是否公开、代码是否可提供、受限数据访问方式。]

## Disclosure and AI Use
[说明利益冲突、资金支持、IRB/RCT（如适用）、AI辅助写作和人工核查责任。]
```

中文期刊版可翻译为“数据与代码可得性”“利益冲突与AI辅助说明”。

## LaTeX 与 PDF 输出规范

除 `ECO_PAPER_DRAFT.md` 外，默认同时生成 `paper.tex`。若系统存在可用 LaTeX 编译器，则进一步生成 `paper.pdf`。PDF 生成失败时，不得中断论文草稿输出；必须保留 `paper.tex` 并在完成汇报中说明缺少的编译器或报错。

### 生成文件

- `ECO_PAPER_DRAFT.md`：可读 Markdown 草稿，保留 profile 配置块和检查清单。
- `paper.tex`：投稿版 LaTeX 源文件，能独立编译。
- `paper.pdf`：若 `xelatex` 或 `latexmk` 可用，则自动编译生成。
- `paper_compile.log`：若编译失败，将错误日志写入该文件。

### 编译器检测顺序

用 Bash 依次检测：

```bash
command -v latexmk
command -v xelatex
```

Windows PowerShell 环境可等价检测：

```powershell
Get-Command latexmk -ErrorAction SilentlyContinue
Get-Command xelatex -ErrorAction SilentlyContinue
```

若 `latexmk` 可用，优先执行：

```bash
latexmk -xelatex -interaction=nonstopmode -halt-on-error paper.tex
```

若仅 `xelatex` 可用，至少执行两遍以解析目录和引用：

```bash
xelatex -interaction=nonstopmode -halt-on-error paper.tex
xelatex -interaction=nonstopmode -halt-on-error paper.tex
```

若两者都不可用，输出：

> 已生成 `paper.tex`，但未生成 PDF：当前环境未检测到 `latexmk` 或 `xelatex`。请安装 TeX Live 或 MiKTeX 后重新运行本技能。

### paper.tex 模板要求

`paper.tex` 必须使用 XeLaTeX/Tectonic 友好的中文模板。国内模式使用中文期刊排版：A4、宋体/ctex默认中文字体、小四或五号、1.5倍行距、一级标题使用“一、二、三”，二级标题使用“（一）（二）（三）”，表格使用统一字号的三线表。

```latex
% !TEX program = xelatex
\documentclass[12pt,a4paper]{article}
\usepackage[UTF8,zihao=-4]{ctex}
\usepackage[top=2.5cm,bottom=2.5cm,left=3cm,right=3cm]{geometry}
\usepackage{setspace}
\onehalfspacing
\usepackage{amsmath,amssymb,bm}
\usepackage{booktabs,threeparttable,longtable,tabularx}
\usepackage{graphicx,caption,subcaption,float}
\usepackage{enumitem}
\usepackage{indentfirst}
\setlength{\parindent}{2em}
\captionsetup[table]{skip=3pt}
```

字体策略：
- 不强制写死 SimSun/SimHei，除非用户环境已确认有这些字体。
- 使用 `ctex` 默认字体配置，保证跨 Windows/Mac/Linux 尽量可编译。
- 如必须指定字体，先检测字体存在，否则保留默认。

### Markdown 到 LaTeX 的转换规则

- 标题、摘要、关键词、JEL、正文8节、参考文献、披露声明都必须同步写入 `paper.tex`。
- 国内模式下，应同步写入国内结构：引言；政策背景、制度事实与理论分析；研究设计；实证结果分析；机制检验；异质性分析；进一步分析；研究结论与政策启示。
- 表格优先引用已有 `tables/*.csv` 或 `tables/*.txt`，不要直接粘贴不可读的软件输出。
- 主文表格必须统一五号或小五号字体、统一 `\tabcolsep`、统一三线表；表内英文变量必须翻译或解释，不得裸放 `ln_tfp`、`CS-DiD Simple ATT`、`Region=East` 这类程序输出。
- 回归系数列必须根据 p 值或 t 值添加显著性星号，表下注明显著性水平。
- 图形使用相对路径，例如 `tables/figure1_event_study.pdf` 或 `.png`；若图文件不存在，在 TeX 中写入占位说明，不编造图。
- 方程使用 `equation` 环境，只编号正文会引用的方程。
- 参考文献如果没有 `.bib`，先在 TeX 末尾使用 `enumerate` 或 `thebibliography`；国内模式编号必须显示为半角方括号 `[1]`、`[2]`，不得使用中文全角括号或【】；不得生成空 bibliography。
- 不要把 `PAPER_PROFILE` 注释块写入正式 `paper.tex` 正文。

### 编译后检查

若 PDF 成功生成，完成前检查：
- `paper.pdf` 文件存在且大小大于 50KB。
- 编译日志中没有 `Fatal error`、`Undefined control sequence`、`File not found`。
- 如果存在 `Citation undefined` 或 `Reference undefined`，再运行一次编译；仍存在时在完成汇报中提示。
- 统计正文中文字符数，必须排除摘要、表格、图、公式、参考文献和附录；低于10,000字时删除或不生成最终版标记，回到分节扩写。
- 检查所有主文表格字号一致；若某张表因列数过多导致过宽，优先缩短中文列名、把英文变量移入表下注释、移动到附录，而不是单独缩小该表字体。
- 检查参考文献编号显示为 `[1]`、`[2]` 格式。

## 写作通用要求

- **语言：** 中文撰写；变量名、模型名如保留英文，必须在首次出现处或表下注释中解释中文含义
- **因果语言：** 避免"相关"，使用"影响"、"导致"、"使X增加/减少Y%"
- **数字密度：** 每段至少出现一个定量描述，不得出现"显著正向"而不给具体数值
- **量级换算：** 系数÷对照组均值=百分比变化，每次报告系数均需换算
- **节奏控制：** 国内模式最低10,000中文正文字符，默认10,000-15,000；CSSCI/高质量实证论文建议12,000-18,000；国际模式9,000-12,000中文字符；若目标期刊另有要求，按目标期刊调整
- **表格引用：** 每张主文表格和图形至少提及一次。国内模式按“理论预期-实证证据-经济含义-政策指向”解释；国际模式按“威胁-证据-解释”解释
- **主文自洽：** 核心数据来源、变量定义、识别假设、主结果和关键稳健性必须在主文中出现，不能只放附录
- **不造事实：** 若状态文件没有提供数据来源、URL、系数或文献页码，写成“待补充/需核查”，不得编造
- **政策语境克制：** 国内模式必须写政策背景，但不得写成宣传稿；政策表述必须回到研究问题、理论机制或政策启示
- **参考文献真实：** 不为凑够20-35篇而虚构文献；缺少中文文献时输出“需补充中文文献清单”，并列出检索关键词
